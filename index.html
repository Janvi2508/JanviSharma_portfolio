Projects

Emotion Detection from Eye Images

Problem: Traditional emotion detection relies on full-face images, which may not always be available or reliable.

Approach: Developed multiple deep learning models including ResNet50, VGG16, MobileNetV2, and DenseNet121 using eye-region images. Applied preprocessing, augmentation, and class balancing techniques.

Result: Achieved 76.81% accuracy. VGG16 demonstrated strong generalization performance.

Chronic Kidney Disease Classification (ViT-B16 + ChOA)

Problem: Early detection of CKD using CT scan images for better clinical decision-making.

Approach: Implemented Vision Transformer (ViT-B16) with 10-fold cross-validation. Applied Chimp Optimization Algorithm for feature selection, reducing 768 features to 508.

Result: Achieved 98.87% accuracy with improved efficiency and reduced computational cost.

Diabetic Retinopathy Detection using Transformers

Problem: Automated detection of retinal diseases from medical images.

Approach: Implemented multiple transformer architectures including ViT, Swin, Twins, and CvT-13.

Result: CvT-13 achieved highest accuracy of 97.8% with superior precision and recall.
